# AvianTech-Using-YOLOv11-and-BirdNET-for-Bird-Monitoring
This automated monitoring system uses YOLOv11 and BirdNET on a Raspberry Pi 5 to provide real-time visual and acoustic bird species identification at the edge. By enabling continuous, internet-independent data collection in remote areas, it offers a scalable and efficient solution for biodiversity conservation and ecological research.
📖 Description

This project presents the design and development of an intelligent, low-cost avian monitoring system that integrates computer vision and bioacoustic analysis for real-time bird detection and classification. The system leverages a Raspberry Pi 5 as an edge computing device, enabling on-site processing without reliance on continuous internet connectivity.

The visual component utilizes a trained YOLOv11 deep learning model to detect and identify bird species from image and video inputs captured through a camera module. Complementing this, the audio component employs BirdNET for automated bird sound classification using environmental audio captured via a microphone sensor (e.g., ReSpeaker 2-Mic HAT).

By combining these two modalities—vision and sound—the system improves detection accuracy and provides a more comprehensive assessment of avian biodiversity. The integrated system is designed for deployment in outdoor environments, particularly in ecologically significant areas such as Iligan City, Philippines.

🎯 Objectives
To develop a real-time bird detection system using computer vision (YOLOv11)
To integrate BirdNET for acoustic-based bird species classification
To implement a Raspberry Pi-based edge computing system for autonomous monitoring
To evaluate the system’s performance in terms of detection accuracy and reliability
⚙️ Key Features
📷 Real-time bird detection using camera input
🎤 Audio-based species identification using BirdNET
🧠 Edge AI processing using Raspberry Pi 5
🌿 Designed for outdoor biodiversity monitoring
🔌 Low-power, standalone system

🌏 Application

This system supports biodiversity conservation efforts by providing researchers and environmental agencies with an automated tool for monitoring bird populations. It contributes to ecological studies, species conservation, and environmental impact assessments.
